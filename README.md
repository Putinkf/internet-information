# The Void Index (Paradox-01)

> «Пустота — это не отсутствие данных. Это их идеальный порядок».

## О проекте
The Void Index — это концептуальный веб-арт и технический эксперимент по исследованию пределов индексации поисковых систем. Визуально проект представляет собой абсолютный минимализм, скрывающий под собой массив данных объемом более 200 МБ.

Проект создан для демонстрации того, как современные алгоритмы (SGE, Googlebot) взаимодействуют с огромными пластами неструктурированной информации, скрытой за «чистым листом».

## Технический стек
* **Core:** HTML5 / CSS3 (Pure Minimalism)
* **Payload Generation:** Python 3.14 (Scrapers & Generators)
* **Architecture:** Zero-UI Data Matrix
* **Deployment:** GitHub Pages

## Особенности
- **Цифровой камуфляж:** Весь SEO-контент вынесен далеко за пределы видимой области и оптимизирован для чтения поисковыми роботами, а не людьми.
- **Стресс-тест браузера:** Несмотря на визуальную пустоту, страница требует значительных ресурсов оперативной памяти для рендеринга DOM-дерева.
- **Инфо-аномалия:** Использование специфических технических ключей (Fedora 43, eBPF, Fabric/Minecraft, Stoicism) для создания уникального отпечатка в поисковой выдаче.

## Как это работает?
1. Скрипт `build.py` собирает данные из различных нейросетевых источников (Gemini, Kimi, Claude).
2. Контент упаковывается в скрытый слой `.seo-layer`.
3. Поисковые системы индексируют «пустоту», создавая информационную аномалию.
